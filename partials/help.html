Navigation - Home screen

Tap on the left top left corner to launch the side bar navigation. When you are in the home screen, tap the top right corner icon to login and logout For non homescreen pages, the arrow icon displayed at the top right corner will take you back to the previous page.

Swipe right to navigate to the next page apps. Swipe left to navigate to the previous page.

Report an issue

If you encounter any issues with the app, please use the 'Report Issue' menu under the sidebar to report the same in an email. Please attach screenshots if any.

Troubleshooting

If you encounter issues related to the app icons not loading after the metadata update, use Troubleshooting option under settings menu from the sidebar. Press the 'Reset My App' button to reset it to default state. Note: You will get an update manager popup after a reset, press 'Yes' to update the app. During this process do not minimize the app, as it can corrupt the metadata. If the metadata gets corrupted then you need to reset it again.
